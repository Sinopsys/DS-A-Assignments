Detailed Description

template<typename T>
class BidiLinkedList< T >

Declares a generic purpose bidirectional list.

Since there are reverse links presented, a list can be traversed both in direct and reverse direction.

Requirements to a T are as follows:

  • T should be default constructable
  • T should be copyable
