Подробное описание

B-дерево, основанное на файловом потоке.

Конкретизирует понятие дерево на случай использования файла для хранения.

Конструктор(ы)

xi::FileBaseBTree::FileBaseBTree ( )

Конструктор по умолчанию.

Для "открытия" дерева необходимо использовать метод open().

xi::FileBaseBTree::FileBaseBTree ( UShort  order,
UShort  recSize,
IComparator *  comparator,
const std::string &  fileName 
)

Конструирует новое B-дерево со структурой, определяемой переданными параметрами.

Создает новое дерево и записывает его в файл fileName. Если файл существует, он перезаписывается. Если файл не может быть открыт, генерируется исключительная ситуация.

Конструктор эквивалентен созданию объекта с параметрами по умолчанию с последующим открытием методом open().

xi::FileBaseBTree::FileBaseBTree ( const std::string &  fileName,
IComparator *  comparator 
)

Конструирует дерево на основе существующего файла B-дерева.

Если файл не может быть открыт, прочитан или содержит неверную структуру, кидает исключительную ситуацию.

xi::FileBaseBTree::~FileBaseBTree ( )

Деструктор.

Не забывает закрыть открытые файловые потоки.
