template<typename Element, typename Compar = std::less<Element>>
class xi::RBTree< Element, Compar >::Node

Узел КЧД.

Большая часть элементов класса является закрытой для внешнего мира и доступной только для самого узла и его потомков. Это сделано с целью инкапсуляции, а само дерево объявлено по отношению к данному классу дружественным, чтобы оно имело доступ к своим узлам.
